# 🚀 IoT Dashboard Project

This project is a complete IoT web application featuring user authentication and a real-time MQTT dashboard with a modern animated UI.

## 🔥 Features

* 🔐 User Signup & Login (MongoDB)
* 🌌 Animated Star Background UI
* 📡 MQTT Dashboard for IoT Communication
* ⚡ Real-time connection status
* 🎨 Clean and modern frontend design

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Protocol: MQTT (WebSockets)

## 📂 Project Structure

* `index.html` → Login Page
* `signup.html` → User Registration
* `dashboard.html` → IoT MQTT Dashboard
* `server.js` → Backend API

## 🚀 How to Run

1. Install dependencies:

   ```
   npm install
   ```

2. Start server:

   ```
   node server.js
   ```

3. Open:

   ```
   index.html
   ```
